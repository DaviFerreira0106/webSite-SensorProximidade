O código implementa um sistema de monitoramento reativo baseado na detecção de movimento, onde um sensor PIR identifica variações de infravermelho no ambiente, acionando um LED e um relé sempre que movimento é detectado. Esse comportamento simula sistemas de segurança ou automação, permitindo a ativação de dispositivos conforme a necessidade. Para evitar oscilações constantes, há um pequeno atraso após a detecção, garantindo estabilidade no funcionamento. Além disso, mensagens são exibidas no console para fornecer um feedback contínuo, tornando o sistema mais transparente e fácil de monitorar.

O código implementa um sistema de medição de distância baseado em um sensor ultrassônico, onde um sinal é enviado e o tempo de resposta é usado para calcular a distância até um objeto. Com base nessa distância, a intensidade de um LED PWM é ajustada, simulando um efeito de proximidade onde a luz se torna mais intensa à medida que o objeto se aproxima. Além disso, um relé é ativado quando a distância é menor que um determinado limite, permitindo a automação de dispositivos conforme a proximidade de um objeto. Esse sistema pode ser aplicado em diversas áreas, como robótica, automação residencial e sistemas de alerta.
